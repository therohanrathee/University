A simple collection of fundamental graph algorithms implemented in Python. This repository is useful for learning and understanding basic graph concepts and their applications.

## 📂 Files

* `adjacency_matrix.py` – Graph representation using adjacency list and matrix
* `bfs_dfs_traversal.py` – Breadth First Search (BFS) and Depth First Search (DFS)
* `topological_sort.py` – Topological Sorting using DFS
* `dijkstras_algorithm.py` – Dijkstra’s shortest path algorithm
* `bellman_ford.py` – Bellman-Ford algorithm (supports negative weights)
* `prims_mst.py` – Prim’s algorithm for Minimum Spanning Tree (MST)

## 🚀 Features

* Clean and simple Python implementations
* Beginner-friendly code
* Covers important graph algorithms

## ⚙️ Requirements

* Python 3.x
* Uses only built-in libraries (`heapq`, `collections`)

## ▶️ How to Run

Clone the repository and run any file:

```bash
python filename.py
```

Example:

```bash
python dijkstras_algorithm.py
```

## 📌 Output

Each script prints results such as:

* Traversal order (BFS/DFS)
* Shortest path distances
* Topological order
* Minimum spanning tree cost

## 📖 Purpose

This project is created for educational purposes to understand core graph algorithms in a simple way.
