## 📌 Overview

This repository contains basic implementations for fundamental algorithm analysis concepts such as time complexity, space complexity, searching algorithms, and recursion.

## 📂 Files

* `time_space_complexity.py` – Demonstration of O(1), O(n), O(n^2), and O(log n) complexities.
* `linear_vs_binary_search.py` – Comparison of Linear Search and Binary Search execution time and memory usage.
* `fibonacci_factorial_dp.py` – Calculation of Factorial, and Fibonacci sequences using recursion vs Dynamic Programming.
* `recursion_time_complexity.py` – Comparing different recursive relations T(n)=T(n/2)+n vs T(n)=2T(n/2)+n.
* `complexity_plot.py` – Script to generate theoretical time complexity graphs.
