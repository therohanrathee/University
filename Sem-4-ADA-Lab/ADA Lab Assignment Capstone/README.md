This repository contains implementations of core algorithms used in optimizing delivery systems, including route planning, parcel selection, and cost minimization.

## 📂 Project Files

```bash
model.py         # Delivery locations, parcels, and vehicle constraints
tsp_rec.py       # TSP using recursion (Backtracking)
greedy.py        # Parcel selection using Greedy approach
knapsack_dp.py   # Profit maximization using Dynamic Programming
dijkstra.py      # Shortest path from warehouse
mst.py           # Minimum Spanning Tree (Prim’s Algorithm)
tsp.py           # TSP using brute-force (permutations)
```

## 🧠 Concepts Covered

* Graph Representation (Matrix & Adjacency List)
* Travelling Salesman Problem (TSP)
* Greedy Algorithms
* Dynamic Programming (0/1 Knapsack)
* Shortest Path Algorithms (Dijkstra)
* Minimum Spanning Tree (Prim’s Algorithm)

## ⚙️ Requirements

* Python 3.x
* Built-in libraries only (`heapq`, `itertools`, `sys`)

## ▶️ How to Run

Run any file using:

```bash
python filename.py
```

Example:

```bash
python dijkstra.py
```

## 📊 Output

Each program prints results such as:

* Minimum delivery route cost
* Maximum profit under capacity constraints
* Shortest distance from warehouse
* Minimum spanning tree cost

This project demonstrates how different algorithmic techniques can be applied to solve real-world logistics and delivery optimization problems.
