## 🧠 Algorithms Implemented

### 1. Crew Assignment using Backtracking

* Assigns crew members to flights
* Ensures no overlapping schedules
* Uses recursion and constraint checking

---

### 2. Branch and Bound – Knapsack Problem

* Solves 0/1 Knapsack efficiently
* Uses bounding to prune unnecessary branches
* Maximizes profit within weight limit

---

### 3. Naive String Matching

* Basic pattern searching technique
* Compares pattern with every substring
* Simple but less efficient

---

### 4. KMP (Knuth-Morris-Pratt) Algorithm

* Efficient string matching algorithm
* Uses LPS (Longest Prefix Suffix) array
* Avoids redundant comparisons

---

### 5. Performance Analysis

* Measures execution time of string matching
* Demonstrates efficiency differences
